# Taura Birra - Sitio React

Proyecto web para la cervecería Taura de la ciudad de La Plata, construido con React y Vite.

## Requisitos

- Node.js 18+ (recomendado)
- pnpm, npm o yarn

## Instalacion

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abrir el navegador en la URL que muestra la terminal (por defecto http://localhost:5173).
